PROJECT

Haiku Together - App for Short Poetry Sharing (Python, Flask, MySQL, Docker, AWS Elastic Beanstalk)

Haiku Together is a web application for poetry sharing. Users can form a group, post and rate short poetry. The application utilizes MySQL for data storage and is containerized using Docker. Deployed on AWS.

Kigo Finder - Word Search Web Application (HTML, CSS, JavaScript, Bootstrap, jQuery, Python)

Kigo Finder is a web application where a user can search "kigo (seasonal word used for Japanese short poetry)" by season and number of syllables. Used Python (Beautiful Soup) for web scraping to collect word data.

LGBTQ+ Film Data Visualization (Javascript, Python, HTML, CSS, Observable, Tableau, Figma)

Conducted data visualization to explore the presence of LGBTQ+ themes in films. Scraped the IMDb database using LGBTQ+ plot keywords. The collected dataset was cleaned and relevant data was extracted using Python libraries such as pandas, Beautifulsoup, and requests.

Machine Learning Analysis of Covid-19 (Python)

Compared the effectiveness of different public health measures by developing a multiple linear regression model using vaccination and masking data with the maximum daily case and death increase.

Spam Messages Classification (Python)

Used supervised machine learning for natural language processing to classify spam messages.

General Social Survey Analysis (R)

Analyzed the General Social Survey (GSS) data to examine the association between education and attitude toward social issues.

Virtual Queue Web Application (HTML, CSS, JavaScript)

Developed a web application where a user can enter phone number and select queue to join virtually. Used Twilio API to send alert messages to users regarding their position in the queue.

Currency Converter - Windows Desktop Application (C#, .NET, API)

Developed Currency Converter , a desktop application to convert US dollar to Japanese Yen. Used ExchangeRate-API to get the latest exchange rate.

Work-Style Reform Project (Research Design, Data Analysis)

Designed and implemented a survey to analyze workplace conditions as part of the inner-ministry reform team. Based on the results, presented a report to the Minister. *Project summary is written in Japanese
